## Изменения

### 1. Расходы по проектам → столбчатая диаграмма
- **Текущее:** горизонтальный bar chart (`layout="vertical"`, Recharts)
- **Цель:** вертикальная столбчатая диаграмма (column chart)
- **Изменения:** убрать `layout="vertical"`, поменять XAxis/YAxis, повернуть подписи (-25°), перенести на всю ширину
- **Данные:** без изменений, `/dashboard/by-project` топ-8

### 2. Круговая диаграмма по специализациям
- **Текущее:** отсутствует на вкладке «Общее»
- **Цель:** PieChart (donut) по специализациям рядом с диаграммой по подразделениям
- **Данные:** `/dashboard/by-specialization` (уже fetched: `bySpec`)
- **Layout:** grid-2 → «По подразделениям» | «По специализациям»

### 3. Движение персонала → линейный график
- **Текущее:** сгруппированный BarChart (3 бара: активных/принято/уволено)
- **Цель:** LineChart с тремя линиями (active=синяя, hired=зелёная, terminated=красная)
- **Сохранить:** click-to-drill детализация по месяцу (оставить `onClick`)
- **Данные:** без изменений, `/dashboard/movements`
